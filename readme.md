# Gods
This is a dueling card game designed by me, Giacomo Nazzaro. 
In this project you will find an python app that lets you play the game against another human or against an agent, based on Monte Carlo Tree Search.

To run the game simply run:
~~~bash
✗ uv run python3 -m gods_graphical.main --help
RAYLIB STATIC 5.5.0.3 LOADED

 Usage: python -m gods_graphical.main [OPTIONS]

╭─ Options ───────────────────────────────────────────────────────────────────────────────╮
│ --host                -h      TEXT     Server host to connect to for online play        │

~~~
